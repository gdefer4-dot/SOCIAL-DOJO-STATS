# Compteur Facebook Dojo

Tableau de bord Facebook du Dojo Club Vieux-Condé.

## Variables Vercel à ajouter

- `FACEBOOK_PAGE_ID`
- `FACEBOOK_ACCESS_TOKEN`

## Dossiers

- `api/facebook.js` : récupère les abonnés via Facebook Graph API
- `public/index.html` : interface
- `public/style.css` : design
- `public/script.js` : animations et mise à jour
